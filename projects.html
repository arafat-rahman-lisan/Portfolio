CINEX - Movie Ticket Management System

CINEX is a full stack ASP.NET Core MVC application delivering a seamless ticket booking experience with a clean UI and role based admin control.

Key Features

  • Movie Discovery: Now Showing, Coming Soon, Top Rated, Genres
  • Showtime Scheduling: theatre, hall, slot scheduling builder
  • Seat Management: interactive map, real time availability, 2 minute holds
  • Payments: PayPal, bKash, Nagad; PDF invoices
  • Admin Portal: theatres, seats, users, schedules, payments
  • Reports: KPIs, filters, revenue insights

Tech Stack

  • Frontend: Razor, Bootstrap, custom CSS
  • Backend: ASP.NET Core MVC (.NET 8), EF Core
  • Database: SQL Server
  • Security: Identity (role based)
  • Docs: QuestPDF invoices
